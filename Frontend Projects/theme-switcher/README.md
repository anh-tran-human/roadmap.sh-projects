# Theme Switcher with CSS Variables

## Goal

Build a tiny UI that can switch between multiple themes using only HTML and CSS. Keep the interface basic: a row of three theme options and one preview card with a heading, short paragraph, and button.

## Requirements

- [x] At least three themes must be defined.
- [x] No JavaScript is allowed.
- [x] The switcher controls must be visible and keyboard accessible.
- [x] The same set of token names must be defined in every theme, so switching themes does not leave any token undefined.
